# Agent Tooling Smoke

## Purpose

Protects the structured coding-agent helper actions that reduce shell/tool churn for common repo-navigation and edit loops.

## Command

```bash
npm run smoke:agent-tooling
```

## Expected Result

The smoke should pass and confirm the router exposes compact, structured tooling actions for repo search, multi-file reads, project maps, symbol lookup, workspace diffs, summarized checks, capability lookup, context packs, repo briefings, docs updates, and atomic patch application.
