# SISO TUI

This directory tracks SISO terminal UI work separately from general agent/runtime capability work.

- `catalog.md` - current catalog of shipped TUI improvements, active priorities, backlog, owners, and verification commands.
- `render-contract.md` - allowed render surfaces, line budgets, and collapsed-output bans.
- `feature-matrix.md` - SISO vs Codex CLI vs Claude Code vs OpenCode feature matrix and priority gaps.

Primary product surface:

```bash
siso
```

Experimental sandbox:

```bash
siso opentui
```
