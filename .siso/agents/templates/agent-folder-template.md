# Persistent Agent Folder Template

Create:

```txt
.siso/agents/<agent-id>/
  agent.md
  goals.md
  memory.md
  controlled-paths.md
  worklog.md
  changelog.md
  metrics.md
  inbox/README.md
  outbox/README.md
  runs/
```

Use descriptive IDs:

```txt
<domain>-<responsibility>
```

Examples:

- `persistent-agent-system-improver`
- `youtube-transcript-ingestion-builder`
- `siso-command-integration-maintainer`
