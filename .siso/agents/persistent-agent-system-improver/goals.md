# Persistent Agent System Improver Goals

## Current Goal

Build and improve the persistent-agent MVP using current SISO infrastructure.

## Current Objectives

1. Create a file-backed persistent agent/team that can be inspected.
2. Track durable goals, memory, worklog, changelog, controlled paths, and token metrics.
3. Create manual run and inspect workflows.
4. Use this agent/team as the first test case for self-improvement.
5. Keep the design simple until the manual loop proves useful.

## Success Criteria

Shaan can ask:

- What is this agent/team trying to do?
- What did it work on last?
- What did it change?
- What does it remember?
- How have its goals changed?
- What code/files does it control?
- How many estimated tokens has it used?
- What should it do next?

And SISO can answer from durable files.
