# Executive Goals

## Goal: Build a persistent executive agent layer for SISO

Status: active
Priority: high
Created: 2026-05-10

### Why it matters

SISO should become more than a request/response coding assistant. It should maintain durable project context, remember decisions, track tasks, coordinate workers, and recommend next actions across sessions.

### Success criteria

- SISO can reconstruct current project state from durable files.
- SISO can distinguish brainstorms, decisions, tasks, and research leads.
- SISO can run an executive review and produce practical next actions.
- SISO can delegate scoped worker-agent tasks.
- The system remains simple enough to inspect and correct manually.
