# SISO Executive State

Status: MVP state skeleton
Created: 2026-05-10

This folder is the durable filesystem-backed memory for the persistent executive agent MVP.

Purpose:

- track active goals and projects
- record decisions
- hold executive-level tasks
- capture inbox events before they become permanent state
- produce reviews that recommend next actions

This is intentionally simple and inspectable. Markdown is the source of truth for the MVP.
