# Workflow: executive-plan

Purpose: convert inbox items, goals, and project state into decisions, tasks, and project updates.

## Steps

1. Read all new inbox items.
2. Group duplicates and related items.
3. Decide handling for each item:
   - promote to decision
   - promote to task
   - update project state
   - add research lead
   - ask Shaan
   - archive/discard
4. Update durable files.
5. Mark processed inbox items.
6. Summarize what changed.

## Rules

- Every new task needs a next step.
- Every decision needs context and consequences.
- Do not expand scope unless it directly supports an active goal.
- Prefer fewer high-quality tasks.
