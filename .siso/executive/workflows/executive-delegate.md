# Workflow: executive-delegate

Purpose: create scoped worker-agent tasks from executive priorities.

## Delegation template

```md
# Worker Task: <title>

Project:
Owner: worker
Priority:
Type: research | code | review | docs | test

## Objective

## Context files

## Specific instructions

## Acceptance criteria

## Report back with

- summary
- files changed/read
- commands run
- findings
- risks/open issues
```

## Rules

- Delegate only scoped work.
- Include context files.
- Include acceptance criteria.
- Prefer read-only research workers before broad code changes.
- Worker reports should update executive state only after review.
