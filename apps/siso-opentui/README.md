# SISO OpenTUI App

This is the experimental OpenTUI/Solid frontend for SISO.

- `src/opencode/` is for MIT-licensed OpenCode TUI code we adapt/vendor.
- `src/siso/` is the SISO adapter layer: Bifrost, child agents, tools, status, sessions.
- `src/main.tsx` will become the entrypoint used by `siso opentui`.

OpenCode license is preserved in `LICENSE.opencode`.
