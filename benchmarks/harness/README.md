# SISO Harness Benchmarks

This is the measurement layer for making SISO a 10x agent harness.

It benchmarks the harness, not just the model:

- task success
- tool efficiency
- token/context efficiency
- code intelligence
- delegation/subagents
- skill usage
- safety/contracts/drift
- memory/improveability
- operator experience

Start local and deterministic. Add public benchmark adapters later.
