# Agent Harness Benchmark Research

This is the inbox for learning from external agent benchmarks, GitHub projects, Sourcegraph-style code intelligence systems, and public agent harness docs.

Goal: turn outside research into SISO benchmark tasks, scorecard metrics, and build tickets.

## Workflow

1. Add source notes under `sources/` or raw notes under `inbox/`.
2. Normalize useful findings into `external-benchmark-map.json`.
3. Convert relevant findings into local SISO benchmark suites under `benchmarks/harness/`.
4. Add build ideas to `docs/capabilities/registry.json` or `docs/capabilities/ideas.md`.
5. Track implementation/testing through `test-space/`.

## Research categories

- coding patch benchmarks
- terminal/task execution benchmarks
- tool-use/API benchmarks
- web/workflow benchmarks
- context/memory benchmarks
- code intelligence/search benchmarks
- delegation/multi-agent benchmarks
- safety/governance benchmarks
- operator experience benchmarks
